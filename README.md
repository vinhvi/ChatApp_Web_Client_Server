# Phần mềm Chat 
1. Công nghệ sử dụng:
 - Backend: Javasript (NodeJS, Express) , Socket
 - Frontend: ReactNative, React
2. Các bước để chạy:
  2.1 Cài đặt 
    2.1.1 Cài đặt Backend
      npm install
    2.1.2 Cài đặt Frontend
      cd frontend
      yarn install
  2.2 Chạy Project
    2.2.1 Chạy backend:
      npm start
     2.2.2 Chạy frontend:
      yarn start
